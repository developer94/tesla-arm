Network Redirector

Network Redirector je File System Driver (FSD) koji sluzi za I/O transfere preko mreze. Predstavlja deo srednjeg sloja Windows operativnog sistema. Moze funkcionisati u klijentskom i serverskom rezimu, a sluzi za redirekciju klijentskih zahteva na server, omogucavajuci transparentno koriscenje mreznih fajlova (ponistava njihovu razliku naspram lokalnih).

Sastoji se od dve softverske komponente:

  1. FSD u kernel rezimu (SYS fajl) koji omogucava funkcije network redirectora
  2. Dynamic link library (DLL fajl) u korisnickom rezimu, koji omogucava korisnickim aplikacijama pristup mreznim fajlovima.

FSD komponenta barata sa Object Menadzerom, Memory Menadzerom, I/O Sistemom i ostalim kernel sistemima da bi omogucio pristup udaljenim fajlovima. Funckionise u oba pravca:

  • Sa klijentske strane: Drajver dobija klijentske zahteve preko I/O Menadzera, i prosledjuje neophodne informacije preko mreze. Prima infomacije/fajlove nazad od servera, i prosledjuje ih aplikaciji koja je inicirala zahtev.
  • Sa serverske strane: Prima zahtev, prosledjuje ga serveru na obradu, i vraca nazad na klijentsku stranu.

Network Redirector takodje implementira mrezni protocol za pristupanje i pozivanje drugih kernel drajvera, korisnickih aplikacija ili servisa koji imaju implementiranu neku vrstu mreznog protokola ili komunikacionog mehanizma. Najcesce je u pitanju komunikacija preko Transport Driver Interface-a (TDI) koji poziva neki od postojecih stack-ova za mrezne protokole (TCP/IP npr.)

User mode network redirector funkcionise na neznatno visem nivou, i pruza funkcionalnost ekvivalentnu WNet-u (Windows Networking, manje vise sve sta vidite na svom racunaru, a vezano je sa mreznim resursima), preko command line-a pomocu komande net. Na primer, net view \\<ime-racunara> bi bilo ekvivalentno pristupanju resursima tog racunara (izlistavanju foldera na toj lokaciji, stampaca na toj lokaciji, itd.). User mode redirector je povezan sa kernel modom pomocu IOCTL-a (Input and Output Control-a), i ne radi nista sem prosledjivanja high level komandi kernel mode drajveru za dalju redirekciju.
